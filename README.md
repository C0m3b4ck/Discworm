# Discworm
A program for playing .exe and audio files, made in Visual Basic 6.0
<br><b><i>Supports: Windows 98, Windows 2000, Windows NT 4.1, Windows ME, Windows XP, Windows Vista, Windows 7, Windows 8, Windows 10, Windows 11</i></b>

# Principles
* Maximum speed, ease-of-use and efficiency
* Easy installation and portability - no dependencies need to be installed by user
* Overall user convenience and ease of use
* Past, present and future compatibility with Windows

# For Developers
When importing the project into your VB6 codespace, <b><i>make sure to import these dependencies:</b></i>
* <b>Windows Media Player/Microsoft Media Player (wmp.dll)</b>
