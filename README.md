# Discworm
A program for playing .exe and audio files, made in Visual Basic 6.0
<br><b><i>Supports: Windows 98, Windows 2000, Windows NT 4.1, Windows ME, Windows XP, Windows Vista, Windows 7, Windows 8, Windows 10, Windows 11</i></b>
<br>Supports Windows Server: Windows Server NT 4.0, Widows Server 2000, Widows Server 2003, Widows Server 2008, Widows Small Business Server 2008, Widows Server 2012, Widows Server 2012 R2, Widows Server 2016, Widows Server 2019, Widows Server 2022, Widows Server 2025
<br>Supports Windows CE: Windows CE 1.0, Windows CE 2.0, Windows CE 2.1, Windows CE 2.11, Windows CE 2.12, Windows CE 3.0, Windows CE .NET 4.1, Windows CE .NET 4.2
<br><b><i>Note: Windows 95 and Windows 98 requires MSVCVB6.dll runtime DLL to function, which is selected in the installer by default</b></i>

# Principles
* Maximum speed, ease-of-use and efficiency
* Easy installation and portability - no dependencies need to be installed by user
* Overall user convenience and ease of use
* Past, present and future compatibility with Windows

# For Developers
When importing the project into your VB6 codespace, <b><i>make sure to import these dependencies:</b></i>
* <b>Windows Media Player/Microsoft Media Player (wmp.dll)</b>
